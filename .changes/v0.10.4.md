# Changelog v0.10.4

## What's Changed

### Added

- Add support for custom CA certificates in engine container (#6821)
- Introduce `withUnixSocket` for service binding in Go SDK (#6834)
- Add `--output` flag to `dagger call` for saving artifacts directly (#6849)

### Fixed

- Fix race condition in module loading when multiple dependencies are resolved concurrently (#6812)
- Resolve intermittent failures in `dagger run` when TTY is not available (#6818)
- Fix incorrect cache invalidation when using `withEnvVariable` with secret values (#6827)
- Correct path resolution for nested module sources on Windows (#6841)

### Changed

- Upgrade Buildkit to v0.13.2 for improved layer caching performance (#6809)
- Improve error messages when module schema validation fails (#6823)
- Reduce memory footprint of the engine session manager (#6836)

### Dependencies

- Bump `github.com/containerd/containerd` from v1.7.12 to v1.7.14
- Bump `golang.org/x/net` from v0.21.0 to v0.23.0
- Bump `github.com/opencontainers/runc` from v1.1.11 to v1.1.12
